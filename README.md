# ESP32-Based IoT Battery Management System

This project presents an IoT-enabled Battery Management System (BMS) using ESP32 for real-time monitoring and safety of lithium-ion battery packs in electric vehicles.

The system integrates embedded hardware, sensors, and communication modules to ensure reliable battery performance and fault detection.

---

# Research Publication

Published in:

**IEEE International Conference on Data, Energy and Communication Networks (DECoN 2025)**

The research paper is included in this repository.

---

# Features

• Real-time monitoring of battery voltage, current, and temperature  
• Fault detection and safety mechanisms  
• IoT-based communication using ESP32  
• Embedded system for EV battery management  
• Scalable architecture for real-world EV applications
